# NLP Ticket Classification

Classify IT support tickets using Natural Language Processing (NLP) and Machine Learning.

**Project Highlights:**
- End-to-end ML pipeline: preprocessing, training, evaluation
- Text vectorization with TF-IDF
- Logistic Regression for ticket classification
- Model saved for reuse and evaluation
- Sample dataset included for demonstration

**Skills:** Python, Pandas, Scikit-Learn, NLP, TF-IDF, Machine Learning

**Repository structure:**
- `preprocessing.py` — data cleaning
- `train.py` — train and save model
- `evaluate.py` — evaluate model performance
- `tickets.csv` — sample dataset
